# Jacktonkin Tap

## How do I install these formulae?

`brew install jacktonkin/tap/<formula>`

Or `brew tap jacktonkin/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jacktonkin/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
